# Angular Studio — actualización funcional

## Pokédex
- Comparador de dos Pokémon con estadísticas, tipos, altura y peso.
- Favoritos persistentes mediante localStorage.
- Búsqueda avanzada por nombre/número, tipo y favoritos.
- Equipo Pokémon de hasta seis integrantes.
- Acciones rápidas en cada tarjeta.

## CRUD Lab
- Ruta protegida `/crud-lab`.
- Crear, consultar, editar y eliminar registros manuales.
- Campos para Pokémon, número nacional, tipo, región, juego, imagen y notas.
- Persistencia local mediante localStorage.
- Búsqueda sobre registros manuales.

## Validación
- Compilación de producción ejecutada correctamente con `npm run build`.
