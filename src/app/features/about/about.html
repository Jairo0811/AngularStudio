Evolución del proyecto

De proyecto académico a aplicación de portafolio

Angular Studio comenzó durante el período académico 2020-T2 del Diplomado en Programación Web con Angular – MESCyT, impartido por el ITLA dentro del programa BecaSoft y vinculado al respaldo del MESCyT y la iniciativa República Digital. Posteriormente fue ampliado con autenticación, consumo de APIs, Pokédex avanzada, gestión CRUD, herramientas para desarrolladores, preferencias de usuario y automatización de recursos.
